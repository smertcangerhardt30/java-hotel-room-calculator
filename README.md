# java-hotel-room-calculator

### üè® Hotel Room Cost Calculator (Java)

This project is a Java-based hotel room simulation developed to practice fundamental Object-Oriented Programming (OOP) concepts.

**Features:**
* Taking room details dynamically from the user via console input (Room type, price, capacity, rating, etc.).
* Calculating extra fees when the number of guests exceeds the room's maximum capacity.
* Applying dynamic price discounts or surges based on the hotel's rating.
* Adding extra charges for rooms with a sea view.
* Comparing the nightly rates of two different hotel rooms.

**Tech Stack:**
* Java
* OOP Concepts (Classes, Objects, Methods)
* `java.util.Scanner` for console inputs
